将应用程序部署到Azure
===============================

将应用程序部署到Azure
Visual Studio Code可以使用Azure轻松地将应用程序部署到云中，我们还创建了演练来帮助你入门。

无论您是全栈、后端、API开发人员还是DevOps工程师，您都可以找到所需的部署步骤。

部署教程
--------------------
以下教程和主题介绍了通过Visual Studio Code创建应用并将其部署到Azure的不同方法：

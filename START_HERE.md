🐛 问题快速诊断
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 找不到 java 命令？
A: 安装Java 11+ 或添加到系统PATH
   检查: java -version

Q: 没有找到媒体文件？
A: 1. 确认网页确实有音频/视频
   2. 有些媒体动态加载，可能不被识别
   3. 尝试其他网址验证

Q: 下载失败？
A: 1. 检查网络连接
   2. 确认URL可访问
   3. 检查输出目录权限

Q: 如何并发下载？
A: 修改源代码使用线程池，或创建脚本并行调用
